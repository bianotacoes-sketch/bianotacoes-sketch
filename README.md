## 📁 Project Highlights

### 📊 Medallion Architecture Data Pipeline (ETL)
An end-to-end data pipeline implementing a modern medallion architecture using **AWS Glue** and **Databricks**. Raw streaming and batch data is ingested into the **Bronze** layer, cleaned and validated in the **Silver** layer, and aggregated into business-level tables in the **Gold** layer. SSIS and Pentaho are integrated to handle legacy on-premises synchronization.
* *Technologies: AWS Glue, Databricks, Pentaho, SSIS, SQL Server, Parquet, PySpark*

### 🧠 Predictive Modeling & Machine Learning
Developing linear and logistic regression models for business classification and trend forecasting. The project covers the entire data science workflow, from Exploratory Data Analysis (EDA) using Pandas, NumPy, and Seaborn to metric validation and deployment of an interactive model inference dashboard using Streamlit.
* *Technologies: Python, TensorFlow, Pandas, NumPy, Scikit-learn, Seaborn, Streamlit*

### 🌐 System Web Integration & Process Automation
A web application built to automate internal processes. It features a robust Python Flask backend communicating with a Postgres database, integrated with a dynamic, custom frontend (HTML/CSS/JS). The system includes background automation modules (RPA) that consume external REST APIs via the Python Requests library.
* *Technologies: Python, Flask, HTML5, CSS3, JavaScript, Requests, PostgreSQL*
